你这个项目建议不要叫“扫描中间件”，因为后续可以扩展成教育行业扫描助手。

我建议项目命名：

# 项目名称

**EduScan Agent（数育扫描助手）**

英文包名：

```
eduscan-agent
```

定位：

> 基于 Electron + Vue3 的 Windows 本地扫描服务客户端，为网页端系统提供扫描仪设备管理、扫描任务执行、文件上传、本地通信能力。

---

---

这套提示词适合直接丢给 **Claude Code / Cursor / Trae 国际版** 开发。

另外我建议你不要纯 Electron 调 TWAIN，而采用：

```
Electron
+
C# Scanner Service
+
TWAIN
```

因为虹光 AU362 这类高速ADF扫描仪，稳定性会比 Node 直接桥接高很多。你后续如果要卖给学校，这个架构更接近商业扫描软件。
